## üìö About

This repository contains lab materials and code for **DAMG 7374: Gen AI with LLMs** taught by **Professor Kishore Aradhya** at Northeastern University.

As a Teaching Assistant and Data/AI Engineering enthusiast, I've built these labs following **Antonio Gulli's Agentic AI design principles** with a focus on hands-on learning and practical implementations. The labs cover a comprehensive overview of foundational GenAI concepts to advanced agentic systems with LangGraph, Snowflake Cortex integration, and Knowledge Graphs, Context Engineering for AI applications.
